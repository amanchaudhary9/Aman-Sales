⚙️ Ek baar setup karo — AI (GPT-4o-mini) Puter.js se free mein kaam karta hai. Email ke liye Google Apps Script setup karo.
📧 Google Apps Script — Email Sender ✓ Connected
Aapki Gmail se emails jaayengi — bilkul free, unlimited. Sirf ek baar setup karo.
1
Naya tab kholo → script.google.com → apne Google account se login karo → New Project click karo
2
Editor mein jo bhi code hai usse Ctrl+A dabao → Delete karo → phir neeche "📋 Copy Code" click karo → editor mein Ctrl+V paste karo → Ctrl+S save karo → project naam: AmanSalesAI
3
Top-right mein Deploy → New Deployment click karo → gear icon → Web App select karo → Execute as: Me → Who has access: Anyone → Deploy click karo
4
Permission popup aayega → Authorize access → Google account choose karo → "Advanced" click karo → "Go to AmanSalesAI (unsafe)" click karo → Allow
5
Deploy hone ke baad Web App URL milegi (copy karo) → neeche paste karo → Save & Connect → phir Test Email bhejo
Google Apps Script Code — Isse Copy Karo
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    MailApp.sendEmail({
      to:      data.to,
      subject: data.subject,
      body:    data.body,
      name:    data.fromName || "Aman Sales AI"
    });
    return ContentService
      .createTextOutput(JSON.stringify({ success: true }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ success: false, error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
